# Shopping Cart System

A simple and beautiful shopping cart system that allows users to add/remove items and calculate the total bill.

## Features

- **Product Catalog**: Display 4 clothing items with prices
- **Add to Cart**: Click "Add to Cart" to add items
- **Remove Items**: Remove individual items from cart
- **Clear Cart**: Clear all items at once with one click
- **Total Calculation**: Automatic real-time total price calculation
- **Beautiful UI**: Animated gradient background with modern card design
- **Responsive Design**: Works on all screen sizes

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: 
  - Animated gradients
  - Keyframe animations
  - Glassmorphism effects
  - Flexbox and Grid layouts
  - Hover effects and transitions
- **JavaScript**: Vanilla JS for cart functionality

## How to Run

1. Open `index.html` in any modern web browser
2. Start adding items to your cart
3. View your cart and total price update automatically

## JavaScript Functions

- `addToCart(id, name, price)` - Adds item to cart or increments quantity
- `removeFromCart(id)` - Removes item from cart
- `clearCart()` - Empties the entire cart
- `calculateTotal()` - Calculates total price
- `updateCart()` - Updates cart display dynamically
